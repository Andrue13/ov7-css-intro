# HTML, CSS, JS, Git intro

## Git

1. Check version: 

```bash
$ git --version
$ ls -la
$ rm -rf .git/
```
2. Git init a new respository

`$ git init`

3. Index files

```
$ git status
$ git add . (все файлы)
$ git add index.html style.css README.md
```
4. Make a commit

```
$ git status
$ git commit -m "First commit"
```
